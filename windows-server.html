Windows Server Deployment

Een praktijkproject waarin een Windows Server omgeving werd opgezet voor een fictieve KMO inclusief gebruikersbeheer, policies en netwerkservices.

Projectoverzicht

Ik installeerde en configureerde Windows Server 2022 inclusief Active Directory, DNS, DHCP en groepsbeleid voor een gesimuleerde kantooromgeving.

Uitvoering

De serveromgeving werd gevirtualiseerd met Hyper-V. Elke gebruiker kreeg een gepersonaliseerd profiel via GPO’s. Policies voor automatische updates en toegang tot netwerkschijven zijn toegepast.

Resultaten

  • Volledig functionerende domeinstructuur
  • Efficiënt gebruikersbeheer en toegangscontrole
  • Automatische softwaredistributie en updates
  • Stabiele virtuele infrastructuur

Magnum Opus Crypto Locker

In dit project heb ik gekozen voor het onderwerp CryptoLocker als magnum opus. Dit is een type ransomware dat door criminelen wordt gebruikt om bedrijven af te persen en geld te stelen. In mijn magnum opus leg ik uit wat CryptoLocker precies is, hoe het werkt, en vooral hoe bedrijven zich ertegen kunnen beschermen. Daarnaast voer ik een virtuele test uit om de effectiviteit van verschillende beschermingsmaatregelen te demonstreren.

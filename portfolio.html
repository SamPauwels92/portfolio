Cybersecurity

Magnum Opus In dit project werd een geïnfecteerd systeem geanalyseerd en hersteld na een aanval met Cryptolocker-ransomware. We implementeerden een efficiënte back-upstrategie, verwijderden de malware volledig en versterkten de beveiliging met proactieve detectietools, zodat toekomstige aanvallen automatisch worden onderschept voordat ze schade kunnen aanrichten.

Netwerksegmentatie voor KMO dummy project

In dit project werd een bedrijfsnetwerk opgesplitst in meerdere VLAN’s om prestaties te verbeteren en interne beveiliging te versterken. Elk team kreeg een eigen subnet, met geconfigureerde firewalls en access control lists (ACL’s). Hierdoor zijn kritieke systemen beter afgeschermd van gebruikerszones en gasten.

Windows Server en Active Directory Setup dummy project

Voor een middelgrote organisatie werd een Windows Server 2022 omgeving opgezet. Active Directory werd geconfigureerd met gebruikersgroepen, beleidsregels en roaming profiles. Automatische login scripts en netwerkshares zorgen voor efficiënt dagelijks gebruik. Back-ups en failover werden voorzien via Hyper-V replicatie.

Wi-Fi Netwerkoptimalisatie op schoolcampus dummy project

De bestaande Wi-Fi infrastructuur werd geanalyseerd en herwerkt. Access points zijn strategisch herplaatst en geconfigureerd met load balancing, band steering en WPA3-beveiliging. Gebruikersrapporten tonen een sterke verbetering in dekking, snelheid en betrouwbaarheid. De monitoring gebeurt via een cloudgebaseerd controllerplatform.

Linux Server Monitoring met Zabbix dummy project

In deze oefening werd een Zabbix-monitoringserver opgezet op een Ubuntu VPS. Zowel Windows- als Linux-clients werden verbonden om CPU, geheugen, netwerk en services in real-time op te volgen. Waarschuwingen via e-mail en dashboards geven snel overzicht van de systeemprestaties en mogelijke incidenten.

Back-up en restore Plan met Veeam dummy project

Er werd een volledig back-upplan uitgewerkt en getest met Veeam Backup en Replication. Zowel volledige images als incrementele back-ups werden uitgevoerd op externe opslag. De restore-procedure werd succesvol getest voor zowel individuele bestanden als een volledige serverherstel. Documentatie werd opgesteld voor toekomstig gebruik.
